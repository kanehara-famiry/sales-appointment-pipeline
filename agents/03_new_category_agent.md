# Agent ③: New Category Agent — 新規カテゴリ調査エージェント

## 役割
既存カテゴリではカバーできない検索ニーズから新規カテゴリ候補を6件作成する。

## ツールセット
terminal, file, web, browser

## 入力情報
- company_name, company_url, company_summary
- 既存カテゴリリスト（Agent②の出力）

## 実行手順

### Step 1: ギャップ分析
企業のWebサイトから以下を抽出:
- 主力製品・技術の専門用語
- 製造工程や加工方法の専門名称
- 既存カテゴリでカバーされていないトピック
- ニッチ技術や特殊加工などの専門キーワード

### Step 2: 検索需要調査
各候補キーワードについて:
- Google検索結果件数を確認
- 実際に検索されていることを確認
- メトリーにカテゴリがないことを確認
- 月間検索ボリューム推定（ツールがあれば、なければ結果件数で代用）

### Step 3: 選定（6件）
以下の基準で選定:
1. 検索需要がある（Google検索結果が100件以上）
2. メトリーにカテゴリがない or 不十分
3. 対象企業に関連性がある
4. 購買意図の高いキーワード（「〜メーカー」「〜会社」「〜業者」）

## 出力形式（JSON）
```json
{
  "new_categories": [
    {
      "name": "新規カテゴリ名",
      "search_volume": "推定値 or '未取得（理由）'",
      "user_intent": "情報収集/比較検討/購入",
      "target": "想定ターゲット",
      "estimated_monthly_searches": "推定値 or null",
      "priority": "高|中|低",
      "reason": "提案理由"
    }
  ]
}
```
